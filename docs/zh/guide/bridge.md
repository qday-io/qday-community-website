# Abelian 區塊鏈跨鏈橋用戶指引

QDay 跨鏈橋是連接 Abelian 區塊鏈和 QDay 網路的重要工具，讓您可以將 ABEL 代幣跨鏈轉換為 WABEL(包裝 ABEL)代幣，在 QDay 生態系統中使用。

:::tip 準備工作
使用跨鏈橋前，請確保您已經:
- 安裝並設定好 [MetaMask 錢包](/zh/guide/metamask.html)
- 在 MetaMask 中 [新增 QDAY 網路](/zh/guide/metamask.html#新增-qday-網路到-metamask)
- 安裝 Abelian Pro 行動錢包應用程式
- 在 Abelian 錢包中持有足夠的 ABEL 代幣
:::

## 訪問跨鏈橋

1. 開啟瀏覽器，訪問 [ABEL 跨鏈橋](https://abel-bridge.qday.io)
2. 進入跨鏈橋頁面後，您將看到跨鏈兌換介面。

## 連接錢包

1. 在跨鏈橋頁面右上角，點擊 "連接錢包" 按鈕;
2. 在彈出的錢包選擇視窗中，選擇 "MetaMask";
3. MetaMask 會彈出連接請求視窗，確認您要連接的帳戶，點擊 "下一步";
4. 在權限請求頁面，點擊 "連接" 按鈕，授權跨鏈橋訪問您的錢包地址;
5. 連接成功後，頁面右上角會顯示您的錢包地址(簡短格式，如 `0x1234...5678`)。

:::warning 注意
請確保 MetaMask 已切換到 QDAY 網路。如果未切換，跨鏈橋可能會提示您切換網路。
:::

## 輸入跨鏈資訊

### 選擇跨鏈方向

1. 在跨鏈橋介面中，確認跨鏈方向為:
   - **從**: Abelian (ABEL)
   - **到**: QDay (WABEL)
2. 如果方向不正確，點擊中間的切換箭頭圖標進行調整。

### 輸入 ABEL 金額

1. 在 "金額" 或 "Amount" 輸入框中，輸入您想要跨鏈的 ABEL 數量;
2. 您可以點擊 "最大" 或 "MAX" 按鈕，自動填入您在 Abelian 錢包中的全部可用餘額;
3. 介面會自動顯示:
   - 您將收到的 WABEL 數量(扣除手續費後)
   - 預估的跨鏈手續費
   - 預估完成時間

:::tip 提示
建議不要一次性轉移全部餘額，保留一些 ABEL 用於支付 Abelian 網路的交易手續費。
:::

### 輸入 QDay 接收地址

1. 在 "QDay 地址" 或 "Recipient Address" 輸入框中，輸入您的 QDay 錢包地址;
2. 您可以:
   - 手動輸入地址
   - 從 MetaMask 複製當前連接的錢包地址
   - 點擊 "使用當前錢包地址" 按鈕自動填入
3. 仔細核對地址是否正確，錯誤的地址可能導致資產永久丟失。

## 生成二維碼並完成支付

### 生成支付二維碼

1. 確認所有資訊無誤後，點擊 "生成二維碼" 或 "Generate QR Code" 按鈕;
2. 頁面會生成一個包含交易資訊的二維碼，包括:
   - 接收地址(跨鏈橋的 Abelian 地址)
   - 轉帳金額
   - 備註資訊(用於識別您的跨鏈請求)
3. 二維碼下方會顯示詳細的交易資訊，請仔細核對。

### 使用 Abelian 錢包掃描支付

1. 開啟您的 Abelian Pro 行動錢包應用程式;
2. 在錢包主介面，點擊 "發送" 或 "Send" 按鈕;
3. 點擊掃描圖標，使用相機掃描跨鏈橋頁面上的二維碼;
4. 錢包會自動填入:
   - 接收地址
   - 轉帳金額
   - 備註資訊
5. 確認交易資訊無誤後，輸入您的錢包密碼或使用生物識別驗證;
6. 點擊 "確認" 或 "Confirm" 按鈕，發送交易到 Abelian 網路。

:::warning 重要提醒
- 請務必使用跨鏈橋生成的二維碼進行轉帳
- 不要手動修改任何交易資訊
- 確保備註資訊完整，否則可能導致跨鏈失敗
:::

## 等待跨鏈確認

### 交易提交確認

1. 在 Abelian 錢包中提交交易後，返回跨鏈橋頁面;

2. 跨鏈橋會自動檢測您的交易，並顯示交易狀態:
   - **待確認**: 交易已提交到 Abelian 網路，等待區塊確認
   - **確認中**: 正在累積區塊確認數
   - **處理中**: 跨鏈橋正在處理您的跨鏈請求
   - **已完成**: WABEL 已發送到您的 QDay 錢包

### 區塊確認要求

1. Abelian 網路需要 **3 個區塊高度確認** 才能確保交易安全性;
2. 每個區塊的生成時間約為 2-3 分鐘，整個確認過程大約需要 **6-10 分鐘**;
3. 您可以在跨鏈橋頁面查看當前確認進度，例如 "1/3"、"2/3"、"3/3";
4. 也可以點擊交易哈希，在 [Abelian 區塊鏈瀏覽器](https://explorer.abelian.info) 查看詳細的交易資訊。

### 接收 WABEL 代幣

1. 當 Abelian 網路確認達到 3 個區塊後，跨鏈橋會自動將 WABEL 發送到您指定的 QDay 地址;
2. 打開 MetaMask 錢包，切換到 QDAY 網路;
3. 如果您尚未新增 WABEL 代幣，請先 [新增 WABEL 代幣到 MetaMask](/zh/guide/metamask.html#新增-wqday、wabel-代幣);
4. 在 "Tokens(代幣)" 標籤頁中，您應該能看到新增的 WABEL 餘額。

:::tip 提示
如果長時間未收到 WABEL， 請檢查:
- Abelian 交易是否已經獲得 3 個確認
- QDay 接收地址是否正確
- MetaMask 是否已正確新增 WABEL 代幣
- 可以在跨鏈橋的 "交易記錄" 或 "History" 頁面查詢跨鏈狀態
:::

## 查詢跨鏈記錄

1. 在跨鏈橋頁面，點擊 "交易記錄" 或 "History" 標籤;
2. 輸入您的 QDay 錢包地址或 Abelian 交易哈希進行查詢;
3. 記錄會顯示:
   - 跨鏈時間
   - 跨鏈金額
   - 交易狀態
   - Abelian 交易哈希
   - QDay 交易哈希
4. 點擊交易哈希可以跳轉到對應的區塊鏈瀏覽器查看詳情。

## 常見問題

### 跨鏈需要多長時間?

正常情況下，從提交 Abelian 交易到收到 WABEL，整個過程需要 **10-15 分鐘**，包括:
- Abelian 網路確認: 6-10 分鐘(3 個區塊)
- 跨鏈橋處理: 1-2 分鐘
- QDay 網路確認: 1-3 分鐘

### 跨鏈手續費是多少?

跨鏈手續費包括兩部分:
- **Abelian 網路手續費**: 由 Abelian 網路收取，用於處理交易
- **跨鏈橋服務費**: 由跨鏈橋收取，用於維護跨鏈服務

具體費用會在跨鏈介面中顯示，請在確認前仔細查看。

### 如果長時間未收到 WABEL 怎麼辦?

如果超過 30 分鐘仍未收到 WABEL，請:
1. 確認 Abelian 交易已成功上鏈(在 Abelian 瀏覽器查詢)
2. 檢查交易備註資訊是否完整
3. 在跨鏈橋的交易記錄中查詢狀態
4. 聯繫 QDay 技術支援，提供您的交易哈希

### 可以將 WABEL 跨回 ABEL 嗎?

可以。在跨鏈橋介面中:
1. 點擊切換箭頭，將方向改為 "QDay → Abelian"
2. 輸入 WABEL 數量和 Abelian 接收地址
3. 使用 MetaMask 確認交易
4. 等待跨鏈完成

## 安全提醒

:::danger 安全警告
- 請務必通過官方連結訪問跨鏈橋，謹防釣魚網站
- 仔細核對所有地址資訊，錯誤的地址可能導致資產永久丟失
- 建議先進行小額測試，確認流程無誤後再進行大額跨鏈
- 保存好交易哈希，以便後續查詢和追蹤
:::
