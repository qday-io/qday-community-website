# QDay 兌換用戶指引

QDay Swap 是基於 QDay 區塊鏈的去中心化交易平台，讓您可以快速兌換各種代幣，並通過提供流動性賺取手續費收益。

:::tip 準備工作
使用 QDay Swap 前，請確保您已經:
- 安裝並設定好 [MetaMask 錢包](/zh/guide/metamask.html)
- 在 MetaMask 中 [新增 QDAY 網路](/zh/guide/metamask.html#新增-qday-網路到-metamask)
- 在錢包中持有足夠的代幣和 QDAY(用於支付手續費)
:::

## 訪問 QDay Swap

1. 開啟瀏覽器，訪問 [QDay Swap](https://swap.qday.io)
2. 進入頁面後，您將看到兌換介面，包含 "兌換" 和 "流動性池" 兩個主要功能。

## 連接錢包

1. 在頁面右上角，點擊 "連接錢包" 按鈕;
2. 在彈出的錢包選擇視窗中，選擇 "MetaMask";
3. MetaMask 會彈出連接請求視窗，確認您要連接的帳戶，點擊 "下一步";
4. 在權限請求頁面，點擊 "連接" 按鈕，授權 QDay Swap 訪問您的錢包地址;
5. 連接成功後，頁面右上角會顯示您的錢包地址。

:::warning 注意
請確保 MetaMask 已切換到 QDAY 網路。如果未切換，QDay Swap 會提示您切換網路。
:::

## 代幣兌換

### 選擇兌換代幣

1. 在 "兌換" 標籤頁中，您會看到兩個代幣選擇框:
   - **上方**: 您要支付的代幣(From)
   - **下方**: 您要接收的代幣(To)

2. 點擊代幣選擇框，在彈出的代幣列表中選擇您想要兌換的代幣;

3. 常見的代幣對包括:
   - WQDAY ↔ USD8
   - WABEL ↔ USD8
   - WQDAY ↔ WABEL

:::tip 提示
如果您要兌換的代幣不在列表中，可以通過輸入代幣合約地址手動添加。代幣合約地址可以在 [QDay 區塊鏈瀏覽器](https://explorer.qday.io) 查詢。
:::

### 輸入兌換金額

1. 在上方的輸入框中，輸入您想要支付的代幣數量;

2. 系統會自動計算並顯示:
   - 您將收到的代幣數量
   - 兌換匯率
   - 價格影響百分比
   - 最低接收數量(考慮滑點後)
   - 預估的手續費

3. 您也可以點擊 "最大" 按鈕，自動填入您的全部可用餘額。

:::warning 價格影響
如果交易金額較大，可能會對價格產生較大影響。建議將價格影響控制在 3% 以內，以避免不必要的損失。
:::

### 設定交易參數

在進行兌換前，您可以點擊右上角的設定圖標⚙️，調整交易參數:

#### 滑點容差(Slippage Tolerance)

滑點容差是指您願意接受的價格變動範圍。

1. 預設選項:
   - **0.1%**: 適合穩定幣兌換，價格波動小
   - **0.5%**: 適合大多數交易場景(推薦)
   - **1%**: 適合價格波動較大的代幣

2. 自訂滑點:
   - 點擊 "自訂" 輸入框，輸入您想要的滑點百分比
   - 建議範圍: 0.1% - 5%

:::tip 提示
- 滑點設定過低可能導致交易失敗
- 滑點設定過高可能導致您收到的代幣少於預期
- 建議使用預設的 0.5% 滑點
:::

#### 交易截止時間(Transaction Deadline)

交易截止時間是指交易必須在多長時間內完成，超時後交易將自動取消。

1. 預設選項:
   - **10 分鐘**: 適合網路擁堵時
   - **20 分鐘**: 標準設定(推薦)
   - **30 分鐘**: 適合不急於完成的交易

2. 自訂時間:
   - 點擊 "自訂" 輸入框，輸入您想要的分鐘數
   - 建議範圍: 5 - 60 分鐘

### 執行兌換

1. 確認所有資訊無誤後，點擊 "兌換" 按鈕;

2. 如果這是您第一次兌換該代幣，系統會先要求您授權:
   - MetaMask 會彈出授權請求視窗
   - 點擊 "確認" 授權 QDay Swap 使用您的代幣
   - 等待授權交易完成(約 10-30 秒)

3. 授權完成後，再次點擊 "兌換" 按鈕;

4. 在彈出的確認視窗中，仔細核對:
   - 支付的代幣數量
   - 接收的代幣數量
   - 兌換匯率
   - 價格影響
   - 手續費

5. 確認無誤後，點擊 "確認兌換" 按鈕;

6. MetaMask 會彈出交易確認視窗:
   - 檢查 Gas 費用是否合理
   - 點擊 "確認" 提交交易

7. 等待交易完成:
   - 交易提交後，頁面會顯示交易狀態
   - 通常需要 10-30 秒完成
   - 您可以點擊交易哈希，在區塊鏈瀏覽器查看詳情

8. 交易完成後，您的錢包餘額會自動更新。

:::tip 提示
如果交易長時間未完成，可能是 Gas 費用設定過低。您可以在 MetaMask 中加速交易或取消後重新提交。
:::

## 流動性池

### 什麼是流動性池?

流動性池是去中心化交易所的核心機制。通過向流動性池提供代幣對，您可以:
- 賺取交易手續費(每筆交易的 0.3%)
- 獲得流動性提供者(LP)代幣
- 隨時添加或移除流動性

### 查看資金池資訊

1. 點擊頁面頂部的 "流動性池" 標籤;

2. 在資金池列表中，您可以看到所有可用的流動性池，包括:
   - **代幣對**: 例如 WQDAY/USD8
   - **總鎖定價值(TVL)**: 池中鎖定的總資產價值
   - **24小時交易量**: 過去24小時的交易總額
   - **24小時手續費**: 過去24小時產生的手續費
   - **年化收益率(APR)**: 預估的年化收益率

3. 點擊任意資金池，可以查看更詳細的資訊:
   - 池中各代幣的數量和比例
   - 歷史交易數據
   - 手續費收益趨勢

### 添加流動性

#### 選擇代幣對

1. 在 "流動性池" 頁面，點擊 "添加流動性" 按鈕;

2. 選擇您要添加的代幣對:
   - 點擊第一個代幣選擇框，選擇代幣 A
   - 點擊第二個代幣選擇框，選擇代幣 B

3. 常見的流動性池包括:
   - **WQDAY/USD8**: 主要交易對，交易量大
   - **WABEL/USD8**: ABEL 生態交易對
   - **WQDAY/WABEL**: QDay 與 ABEL 的直接兌換

#### 選擇手續費等級

QDay Swap 支援多個手續費等級，不同等級適合不同的代幣對:

1. **0.01%**: 適合穩定幣對(如 USDT/USD8)
   - 價格波動極小
   - 交易量大
   - 收益穩定但較低

2. **0.05%**: 適合相關性強的代幣對(如 WQDAY/USD8)
   - 價格波動較小
   - 交易量中等
   - 平衡風險和收益(推薦)

3. **0.3%**: 適合一般代幣對(如 WQDAY/WABEL)
   - 價格波動較大
   - 交易量較小
   - 高風險高收益

4. **1%**: 適合波動性極大的代幣對
   - 價格波動劇烈
   - 交易量不穩定
   - 最高風險和收益

:::tip 提示
選擇手續費等級時，建議:
- 新手選擇 0.05% 或 0.3% 等級
- 優先選擇交易量大、TVL 高的池子
- 關注年化收益率(APR)，但不要只看收益率
:::

#### 輸入流動性金額

1. 在代幣 A 的輸入框中，輸入您想要添加的數量;

2. 系統會自動計算代幣 B 的數量，以保持池中的比例;

3. 介面會顯示:
   - 您將獲得的 LP 代幣數量
   - 您在池中的份額百分比
   - 當前的兌換匯率
   - 預估的手續費收益

:::warning 重要提醒
添加流動性需要同時提供兩種代幣，且比例由當前池中的價格決定。請確保您的錢包中有足夠的兩種代幣。
:::

#### 確認並添加

1. 確認所有資訊無誤後，點擊 "添加流動性" 按鈕;

2. 如果這是您第一次添加這些代幣，需要分別授權兩種代幣:
   - 先授權代幣 A
   - 再授權代幣 B
   - 每次授權都需要在 MetaMask 中確認

3. 授權完成後，點擊 "確認添加流動性" 按鈕;

4. MetaMask 會彈出交易確認視窗，點擊 "確認";

5. 等待交易完成後，您將收到 LP 代幣，代表您在流動性池中的份額。

### 查看我的倉位

1. 在 "流動性池" 頁面，點擊 "我的倉位" 標籤;

2. 這裡會顯示您所有的流動性倉位，包括:
   - 代幣對名稱
   - 您提供的流動性價值
   - LP 代幣數量
   - 池中的份額百分比
   - 累計賺取的手續費
   - 當前的未實現盈虧

3. 點擊任意倉位，可以查看詳細資訊和進行操作。

### 移除流動性

1. 在 "我的倉位" 中，選擇您想要移除的流動性池;

2. 點擊 "移除流動性" 按鈕;

3. 選擇移除的百分比:
   - **25%**: 移除四分之一
   - **50%**: 移除一半
   - **75%**: 移除四分之三
   - **100%**: 全部移除
   - **自訂**: 輸入任意百分比

4. 系統會顯示您將收到的代幣數量;

5. 點擊 "確認移除" 按鈕;

6. MetaMask 會彈出交易確認視窗，點擊 "確認";

7. 交易完成後，LP 代幣會被銷毀，您將收到對應比例的兩種代幣，以及累計的手續費收益。

:::tip 提示
移除流動性時，您收到的代幣數量可能與添加時不同，這是由於:
- 池中價格比例發生了變化
- 您賺取了交易手續費
- 可能存在無常損失(Impermanent Loss)
:::

## 理解無常損失

### 什麼是無常損失?

無常損失是指當您提供流動性時，由於代幣價格變化導致的潛在損失。

**舉例說明**:
- 您添加 1 WQDAY(價值 $100)和 100 USD8(價值 $100)
- 總價值: $200
- 如果 WQDAY 價格上漲到 $200:
  - 池中比例會自動調整
  - 移除流動性時，您可能得到 0.707 WQDAY 和 141.4 USD8
  - 總價值: $282.8
  - 如果您單純持有: 1 WQDAY($200) + 100 USD8($100) = $300
  - 無常損失: $300 - $282.8 = $17.2

### 如何減少無常損失?

1. **選擇相關性強的代幣對**:
   - 穩定幣對(USD8/USDT)幾乎沒有無常損失
   - 相關資產(WQDAY/WABEL)無常損失較小

2. **長期持有**:
   - 手續費收益可以彌補無常損失
   - 價格最終可能回到原點

3. **選擇高手續費等級**:
   - 更高的手續費收益
   - 但交易量可能較小

4. **密切關注市場**:
   - 價格大幅波動時考慮移除流動性
   - 價格穩定時添加流動性

## 查看交易記錄

1. 在頁面右上角，點擊您的錢包地址;

2. 在下拉選單中，選擇 "交易記錄";

3. 這裡會顯示您所有的兌換和流動性操作記錄，包括:
   - 交易時間
   - 交易類型(兌換/添加流動性/移除流動性)
   - 代幣數量
   - 交易狀態
   - 交易哈希

4. 點擊交易哈希，可以在區塊鏈瀏覽器查看詳細資訊。

## 常見問題

### 為什麼我的交易失敗了?

交易失敗的常見原因:
1. **滑點設定過低**: 價格變動超過了您設定的滑點容差
2. **Gas 費用不足**: 錢包中的 QDAY 不足以支付手續費
3. **流動性不足**: 池中的代幣數量不足以完成您的交易
4. **交易超時**: 交易未在截止時間內完成

**解決方法**:
- 增加滑點容差(建議 0.5% - 1%)
- 確保錢包中有足夠的 QDAY
- 減少交易金額
- 延長交易截止時間

### 為什麼顯示的價格與實際收到的不同?

這是正常現象，原因包括:
1. **價格影響**: 您的交易改變了池中的價格
2. **滑點**: 在交易確認期間價格發生了變化
3. **手續費**: 交易手續費從您的收益中扣除

### 如何獲得更好的兌換價格?

1. **減少交易金額**: 大額交易會產生較大的價格影響
2. **選擇流動性充足的池**: TVL 越大，價格影響越小
3. **避開交易高峰**: 網路擁堵時 Gas 費用會上漲
4. **分批交易**: 將大額交易拆分成多筆小額交易

### 提供流動性安全嗎?

提供流動性存在以下風險:
1. **無常損失**: 代幣價格波動導致的損失
2. **智能合約風險**: 雖然經過審計，但仍存在潛在漏洞
3. **流動性風險**: 極端情況下可能無法立即移除流動性

**建議**:
- 只投入您能承受損失的資金
- 選擇經過審計的流動性池
- 分散投資，不要將所有資金放在一個池中
- 定期檢查您的倉位表現

### 手續費收益如何計算?

手續費收益取決於:
1. **您的份額**: 您在池中佔的比例
2. **交易量**: 池中的交易越多，手續費越多
3. **手續費等級**: 不同等級收取不同比例的手續費
4. **時間**: 提供流動性的時間越長，累計收益越多

**計算公式**:
```
您的收益 = 總手續費 × 您的份額百分比
```

## 安全提醒

:::danger 安全警告
- 請務必通過官方連結訪問 QDay Swap，謹防釣魚網站
- 仔細核對代幣合約地址，避免兌換假幣
- 不要授權不明來源的代幣合約
- 建議先進行小額測試，確認流程無誤後再進行大額交易
- 妥善保管您的私鑰和助記詞，不要與任何人分享
:::
